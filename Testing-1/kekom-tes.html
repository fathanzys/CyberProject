AES (Advanced Encryption Standard)

AES is a symmetric cryptography algorithm used to encrypt and decrypt data using the same key...

Advantages: Fast, efficient, and secure (especially AES 256-bit).
Disadvantages: Requires secure symmetric key distribution.

Sourced from: Wikipedia - AES

RSA (Rivest–Shamir–Adleman)

RSA is a public key cryptography algorithm that uses a public and private key pair for encryption and decryption...

Advantages: Security based on the difficulty of factorizing large numbers, and enables digital signatures.
Disadvantages: Slower than symmetric encryption algorithms like AES and requires larger keys for high security.
